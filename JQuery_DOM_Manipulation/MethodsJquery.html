Dom methods in jquery

append() -- this will append the text to the same html elements but at the end

after() -- the text will be added after the html elements that mean if we select div element to use after method then text is added after the div element

before() -- this will add the text before the html elements and in begining

prepend() -- this will add the text begin of the same html elements
